About This Project

This project was created by Barrett Crusse

Libraries used:

  • Showdown.js
    • Allows parsing markdown code into HTML elements
  • Bootstrap
    • Provides styling and some pre-built components
  • JQuery
    • Enables the creation of listeners for modal opening and closing
  • Platform.js
    • Used to identify the user's platform in order to use either `ctrl` or `cmd` keypresses for shortcuts

APIs used:

  • The Markdown Guide
    • Serves a list of markdown elements, as well as proper syntax and examples
